## 1. 游戏概述

| 项目 | 描述 |
|------|------|
| 游戏名称 | Abyss Walker / 深渊行者 |
| 类型 | 中世纪奇幻 Roguelike |
| 玩法 | 回合制地牢探索 + 自动攻击 |
| 视角 | 俯视角 2D 网格 |
| 平台 | PC + 移动端 |
| 技术栈 | Unity (C#) + Python (PyTorch) |
| 通信 | TCP Socket + JSON |
| 开发策略 | 模块化渐进开发 |

**核心特色:** 玩家扮演深渊行者，在随机生成的地牢中逐层探索。角色自动攻击范围内敌人，玩家专注于走位和技能释放时机的策略决策。每次冒险结束后可获得永久升级。AI Agent 可自主学习玩游戏，敌人 AI 具备智能行为。

## 3. 通信协议

### 消息格式

所有消息采用 **长度前缀 + JSON** 格式:
```
[4字节: 消息长度 (big-endian)] [JSON 字符串]
```

### 消息类型

**Unity -> Python:**

```json
{
  "type": "game_state",
  "timestamp": 1234567890,
  "payload": {
    "player": {
      "hp": 100, "max_hp": 100,
      "pos": [3, 5],
      "attack": 15, "defense": 8,
      "skills": ["slash", "heal"],
      "level": 3, "exp": 120
    },
    "enemies": [
      { "id": 1, "type": "skeleton", "hp": 30, "max_hp": 30, "pos": [7, 2], "attack": 10 }
    ],
    "map": {
      "width": 10, "height": 10,
      "tiles": [[0,0,1,1,...]],
      "rooms": [{"x":0,"y":0,"w":5,"h":5}]
    },
    "floor": 3,
    "events": [{"pos": [5,5], "type": "treasure", "active": true}]
  }
}
```

**Python -> Unity:**

```json
{
  "type": "ai_decision",
  "request_id": "uuid-xxx",
  "payload": {
    "agent_action": { "move": [3, 6], "skill": "slash", "target_id": 1 },
    "enemy_actions": [
      { "id": 1, "move": [6, 2], "attack_player": true }
    ]
  }
}
```

```json
{
  "type": "map_data",
  "payload": {
    "width": 15, "height": 15,
    "tiles": [[...]],
    "rooms": [...],
    "spawn_point": [2, 2],
    "exit_point": [13, 13],
    "enemies": [{"type": "skeleton", "pos": [5,5]}],
    "events": [{"type": "treasure", "pos": [8,3]}]
  }
}
```

## 4. 游戏核心机制

### 4.1 回合制探索

```
回合流程:
1. 玩家回合: 选择移动/技能 -> 自动攻击 -> 触发事件 -> 结算
2. 敌人回合: AI 决策 -> 移动/攻击 -> 结算
3. 环境回合: 检查通关 -> 更新难度 -> 生成下一层
```

### 4.2 自动战斗

- 角色进入敌人 1 格范围内自动触发攻击
- 攻击频率由攻速属性决定
- 玩家可手动释放技能 (消耗能量/冷却时间)
- 优先级: 手动技能 > 自动攻击 > 自动移动

### 4.3 随机事件

| 事件 | 触发 | 效果 |
|------|------|------|
| 宝箱 | 踩格子 | 金币/装备/消耗品 |
| 陷阱 | 隐藏格子 | 伤害/负面状态 |
| 商店 | 每层随机 | 购买/升级 |
| NPC | 特定房间 | 任务/线索 |
| 祭坛 | 每3层 | 增益选择 |
| 精英怪 | 每5层 | 强敌+奖励 |

### 4.4 Meta 进度

每局结束后获得 **深渊之魂** (永久货币):
- 击杀敌人 -> 魂
- 深层探索 -> 倍率加成
- 完成事件 -> 额外魂

永久升级: 基础属性 / 新职业 / 新技能池 / 初始道具

## 5. AI 系统

### 5.1 强化学习 Agent

**观测空间:**
- 玩家状态: [hp, max_hp, atk, def, level, x, y] (7维)
- 地图: 10x10 网格 (100维)
- 敌人信息: 最多5个 [hp, x, y, type] (20维)
- 总计: ~127维

**动作空间:** Discrete(5 + N) — 移动5种 + 技能N种

**奖励函数:**
- 击杀敌人: +10
- 受伤: -5 * (伤害/max_hp)
- 到达下一层: +50
- 死亡: -100
- 每步存活: +0.1
- 拾取物品: +5

**网络结构 (Actor-Critic):**
```
输入 (127) -> FC(256, ReLU) -> FC(256, ReLU)
  ├── Actor: FC(128) -> FC(action_dim, Softmax)
  └── Critic: FC(128) -> FC(1, Linear)
```

**训练:** PPO + GAE, PyTorch 实现

### 5.2 敌人 AI (行为树)

```
Selector
├── Sequence: 攻击 (玩家在范围?) -> 攻击
├── Sequence: 追击 (玩家在视野?) -> A*寻路移动
└── Action: 随机巡逻
```

| 敌人 | 行为 |
|------|------|
| 骷髅 | 直线追击 |
| 哥布林 | 绕路包抄, 会逃跑 |
| 暗影法师 | 远程攻击, 保持距离 |
| Boss | 行为树 + RL 混合, 多阶段 |

### 5.3 地图生成 (BSP)

1. 初始化大矩形
2. 递归二分切割 (40%-60% 位置)
3. 叶子节点生成房间
4. L形走廊连接相邻房间
5. 放置入口/出口/敌人/事件/Boss

## 7. 技术选型

| 层面 | 选择 | 理由 |
|------|------|------|
| 游戏引擎 | Unity 2022+ LTS | 移动端支持最佳 |
| AI 语言 | Python 3.10+ | PyTorch 生态 |
| 深度学习 | PyTorch 2.x | 动态图, 调试方便 |
| RL 框架 | 自实现 PPO + Gymnasium | 教学价值高 |
| 通信 | TCP + JSON (开发) | 先跑通再优化 |
| 序列化 | JSON -> MessagePack (优化) | 渐进升级 |
| 地图生成 | BSP 算法 | 经典可靠 |
| 敌人 AI | 行为树 + RL 混合 | 渐进复杂 |
| 版本控制 | Git + GitHub | 项目要求 |
| 环境管理 | Conda | 项目要求 |
